## 3. Краткая архитектура

**Описание архитектуры:**
Двухзвенная архитектура: фронтенд (статический HTML + Alpine.js) отдаётся через Nginx, который также работает как reverse proxy для FastAPI-бэкенда. Бэкенд предоставляет REST API (версионированное /api/v1/) для CRUD проектов, финансовых расчётов, AI-генерации (описания, риски, анализ), аутентификации (JWT), управления пользователями, экспорта отчётов. Финансовые расчёты дублируются: real-time на фронтенде через vanilla JS и серверная валидация на бэкенде через numpy-financial. AI-функции реализованы через OpenAI API (GPT). Данные хранятся в SQLite (dev) или PostgreSQL (prod). Приложение контейнеризировано через Docker Compose (backend + nginx).

**Основной стек:**

| Слой | Технология |
|------|-----------|
| Backend | Python 3.11, FastAPI, SQLAlchemy 2, Pydantic 2, Uvicorn |
| База данных | SQLite (dev) / PostgreSQL (prod) |
| AI | OpenAI API (GPT) |
| Финансовые расчёты | numpy-financial (backend), vanilla JS (frontend) |
| Аутентификация | JWT (python-jose), passlib + bcrypt |
| Экспорт | WeasyPrint (PDF), openpyxl (Excel) |
| Frontend | HTML5, Alpine.js 3, Font Awesome |
| Веб-сервер | Nginx 1.25 (reverse proxy + static) |
| Контейнеризация | Docker, Docker Compose |
| Email | SMTP (для отправки паролей при регистрации) |

**Схема взаимодействия / ссылка на диаграмму:**

```
┌──────────────┐     HTTP/HTTPS      ┌──────────────────┐
│   Браузер    │ ──────────────────── │   Nginx :80/443  │

                                                  │        │
                                          ┌───────▼──┐  ┌──▼──────────┐
                                          │ SQLite / │  │ OpenAI API  │
```

## 10. Локальное тестирование

**Тестировался ли сервис локально:**

- [x] Да
- [ ] Нет

**Как запускался:**

1. **Docker Compose (основной способ):**
   ```bash
   cp .env.example .env
   # Отредактировать .env: добавить OPENAI_API_KEY
   docker compose up -d
   # Приложение: http://localhost, API docs: http://localhost/api/docs
   ```

2. **Без Docker (для разработки):**
   ```bash
   cd backend
   python -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   uvicorn app.main:app --reload --port 8000
   # Frontend: cd frontend && python -m http.server 3000
   ```

**Какие ресурсы использовались:**

- Внутренние: SQLite (встроенная БД, не требует отдельного сервера)
- Внешние: OpenAI API (для AI-функций), CDN (Alpine.js, Font Awesome)
